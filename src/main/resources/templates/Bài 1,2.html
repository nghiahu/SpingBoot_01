Spring MVC Spring Boot
Khái niệm Là một module của Spring Framework được dùng để xây dựng ứng dụng web theo mô hình MVC Spring Boot là framework Java giúp đơn giản hóa và tăng tốc quá trình phát triển ứng dụng web bằng cách cung cấp cấu hình mặc định và giảm thiểu cấu hình phức tạp.
Cấu hình Cấu hình thủ công với XML hoặc java cofig Tự động cấu hình thông minh (auto-configuration)
Cách khởi tạo dự án Tạo thủ công hoặc thông qua IDE như Eclipse, IntelliJ với cấu hình phức tạp. Dễ dàng với Spring Initializr hoặc dòng lệnh.
Quản lý phụ thuộc Cần thêm từng thư viện Sử dụng các “starter” như spring-boot-starter-web, starter-data-jpa
Thời gian phát triển Chậm hơn do phải cấu hình thủ công nhiều bước. Nhanh chóng vì hầu hết đã được cấu hình sẵn.
